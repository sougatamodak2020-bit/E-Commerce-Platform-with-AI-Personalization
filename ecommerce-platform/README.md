# MAISON ELITE - Premium E-Commerce Platform

## Quick Start

1. Install dependencies: `cd frontend && npm install`
2. Setup Supabase database (run SQL from database/schemas/)
3. Copy .env.example to .env.local and add your keys
4. Run: `npm run dev`
5. Visit: http://localhost:3000

## Default Credentials
- Admin: admin@maisonelite.com / Admin@123
- Demo: demo@maisonelite.com / Demo@123
